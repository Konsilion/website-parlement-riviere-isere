# Ressources

Le Parlement de la rivière Isère rassemble une documentation riche sur les enjeux de l'eau, les droits de la nature et la démarche fluvio-sensible. Cette rubrique met en avant les documents principaux que nous souhaitons partager.

!!! info "Banque de données"
    Nous disposons d'une banque de données considérable que nous partageons avec plaisir à qui le souhaiterait. N'hésitez pas à nous contacter pour en savoir plus : [parlement-isere@gresille.org](mailto:parlement-isere@gresille.org)
